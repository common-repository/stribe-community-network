== Description ==

Stribe Community Network allows you to easily plug a Stribe.com community in your Wordpress blog.
Stribe is a free meta social network for blogs to get more more exposure and traffic by sharing social interactions.

See http://www.stribe.com/blog for more information, real-life examples and to get your free Stribe API key.

== Installation ==

1. Uncompress `stribe-community-network.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your account on www.stribe.com to add your blog and get a Stribe API key
4. Go to the Settings section and fill in your API key.
5. Go to your widget section and drag and drop the Stribe widget to your sidebar

It's done! Easy, uh?

== Frequently Asked Questions ==

= What is Stribe? =

Stribe allow websites sharing the same interests to provide social networking tools to a common community of visitors, thus bringing more value and traffic to the "stribed" blogs.
Stribe can appear in your blog like a widget in your sidebar, or like a social toolbar.
More information at http://www.stribe.com/blog

= What are the Stribe features? =

Stribe provides public and private chat in a real-time newsfeed, private messaging and push to Facebook and Twitter in addition to cross-blogs interactions, invitations and link sharing.
Stribe is cool!
